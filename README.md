# PyPhot

一个用于光纤仿真运算的 Python 库。

## 运行代码

```shell
# 安装 PyPhot
pip install phot

# 运行代码示例
python3 example.py
```

主目录下的 Python 文件 [example.py](example.py) 展示了如何使用我们的 API，请仔细阅读。
